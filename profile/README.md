### AI & Agent Systems

| Repository | What it does |
|:-----------|:-------------|
| [`memory-service`](https://github.com/goroshi-hq/memory-service) | Per-user persistent memory for AI agents. Vector search with pgvector, lifecycle management, consolidation. |
| [`rag-engine`](https://github.com/goroshi-hq/rag-engine) | Retrieval-augmented generation pipeline. Document ingestion, semantic chunking, embedding, vector search. |
| [`agent-mesh`](https://github.com/goroshi-hq/agent-mesh) | Multi-agent orchestration. Shared state, task routing, failure recovery, typed message passing. |

### Infrastructure

| Repository | What it does |
